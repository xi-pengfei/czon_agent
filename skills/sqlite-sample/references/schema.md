# Sample Database Schema

## Table: employees

| Column     | Type    | Description             |
|------------|---------|-------------------------|
| id         | INTEGER | Primary key             |
| name       | TEXT    | Employee full name      |
| department | TEXT    | Department name         |
| salary     | REAL    | Monthly salary (CNY)    |
| hire_date  | TEXT    | Hire date (YYYY-MM-DD)  |

## Sample Departments

- 工程部
- 产品部
- 市场部
- 财务部
- 人力资源部
